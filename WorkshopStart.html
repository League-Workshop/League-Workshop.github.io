Survey / Github

Ask students to complete the pre-workshop survey: bit.ly/LeagueBeforeWorkshop

(Use this activity to establish if all students have set up a Github account)

Attendance and Photos

In the LeagueHub (Pike13) mark attendance for each student, take a head shot photo and upload it into their client profile.

Warm-up

While students are still arriving and completing admin tasks, give the class the following warm-up activities:

bit.ly/blockly-maze

Clone Workshop Code

           Students who have GitHub accounts should login at gitbub.com, then get the GitHub classroom assignment for the Workshop at:

bit.ly/introtojava

           They will first need to allow GitHub classroom to access their account.

           Then follow the prompts to accept the assignment from the classroom.

           They should now have a copy of the LeagueWorkshop/intro-to-java-workshop-XXXXXXX (where XXXXXXX is their github id)

           Finally they will clone the code to their Desktop (either push from github.com, or go to Github Desktop and clone from there).

           NOTE: There is no need to import this code into Eclipse at this time. The first session is Processing based.
