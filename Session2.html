Intro To Java Workshop - Section 2

Introduction to Eclipse

           If any student did not clone the workshop code in Section 1, they need to do that now.

           Import the cloned Java Project into Eclipse. It should be on the Desktop (see Section 1).
                      - have students find the "MyFristJavaProgram" code and talk about the content.

           Introduce the Robot and how to create one and give it a name.
                      - have students make a Robot. Run the program.

           Introduce the idea of listing the things a robot can do (move, turn, sparkle, hide, etc.)
           Talk about the Robot's pen and how to make it draw.
                      - have students play around with their Robot. Looking up commands and running the code to see what happens.
                      - now have the students write code to draw a square.
