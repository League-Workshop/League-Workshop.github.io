Intro To Java Workshop - Section 4

If any student did not clone the workshop code, they should do that now and put it on the Desktop (see Section 1).



Text-Based Games (Eclipse)

Some of these games use Random numbers which will need to be explained on the white board for most students.

            Quiz Game (Dialogs)

                      Ask user questions, check answers, keep a score. Less difficulty.

            Awesome or Not (Random and Dialogs)

                      Generate different responses based on a random number. Less difficulty.

            Dragon Fight (Random and Dialogs)

                      Use random numbers for game flow. Keep totals for player and dragon health. Medium difficulty.



Robot-Based Game (Eclipse)

            Robot Treasure Hunt (xy coordinates, syso)

                      Requires attention to details in written instructions.
                      Program locks if students use wrong Robot methods.
                      Requires access to Youtube. Medium difficulty.



Visual Games (Processing)

            Terminator Cat (xy coordinates, loops)

                      Needs an image. Cat shoots lasers out of its eyes. Medium difficulty.



            Fish Feeder (xy coordinates, loops)

                      Fish catches falling food. Lots of optional extras. Most difficulty.
