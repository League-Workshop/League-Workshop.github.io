If any student did not clone the workshop code, they should do that now and put it on the Desktop (see Section 1).



Scary Maze(Eclipse)

The recipe advises students to use Pixlr to make a maze, but any simple image editor will work. If time is short, a pre-drawn maze is provided in the package, or students can find a suitable image on the Internet to drag and drop into the project.

Note: Explain to the students that the recipe uses color to determine which is the maze path and which is "out of bounds". If they want to use their own image, they need one that has distinct color zones.

Alternative
